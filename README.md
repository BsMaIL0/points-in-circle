# Точки в окружности

## Задача

Дана окружность с целочисленным радиусом R (0 < R < 2·10^9).  
Необходимо посчитать количество точек с целочисленными координатами (x, y), лежащих **внутри** или **на границе** окружности.

## Входные данные

- Одно целое число R — радиус окружности.

## Выходные данные

- Одно целое число — количество точек с целочисленными координатами, удовлетворяющих условию x² + y² ≤ R².

## Ограничения

- 0 < R < 2 000 000 000
- Тип данных: unsigned long long (64 бита) для предотвращения переполнения

## Алгоритм

1. Если R = 0 — ответ 1 (только точка (0,0)).
2. Иначе:
   - Рассматриваем первую четверть (включая оси, но без центра).
   - Для каждого y от 1 до R-1 находим максимальное x такое, что x² + y² ≤ R².
   - Суммируем количество точек в четверти.
3. Итоговая формула:

   total = count_quarter * 4 + R * 4 + 1

   - *4 — четыре четверти
   - + R*4 — точки на осях (кроме центра)
   - + 1 — центр (0,0)

## Пример

**Вход:**

2

**Выход:**

13


Пояснение: точки \((-2,0), (-1,-1), (-1,0), (-1,1), (0,-2), (0,-1), (0,0), (0,1), (0,2), (1,-1), (1,0), (1,1), (2,0)\) — всего 13.
